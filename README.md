# JavaScript Tutor

Εκπαιδευτικό λογισμικό για το Θέμα 2: εκμάθηση γλώσσας προγραμματισμού.

## Εκτέλεση

```powershell
python server.py
```

Άνοιγμα στον browser:

```text
http://127.0.0.1:8000
```

## Κάλυψη απαιτήσεων

- Εννέα ενότητες JavaScript με εμπλουτισμένη θεωρία, παραδείγματα κώδικα, όρους-κλειδιά και μαθησιακά αποτελέσματα.
- Οδηγός βήμα προς βήμα ανά ενότητα: μελέτη, εξάσκηση, project, quiz.
- Διαφορετικές μορφές εξάσκησης ανά ενότητα: αντιστοίχιση, σενάρια, debugging, code tasks, trace ασκήσεις και mini labs.
- Dropdown course navigator και tabs για γρήγορη επιλογή ενότητας.
- Quiz αυτοαξιολόγησης με 12 ερωτήσεις ανά ενότητα.
- Επαναληπτικό quiz με 36 ερωτήσεις από όλες τις ενότητες.
- Διαδραστικές δραστηριότητες εξάσκησης με άμεσο έλεγχο και ενδεικτικές λύσεις.
- Mini project και πρακτικές εργασίες ανά ενότητα.
- Σύνδεση/επιστροφή χρήστη με email ώστε να συνεχίζει στο ίδιο προφίλ.
- SQLite βάση δεδομένων για χρήστες, email σύνδεσης, επισκέψεις, προσπάθειες quiz και απαντήσεις.
- Καταγραφή δραστηριοτήτων εξάσκησης στη βάση δεδομένων.
- Αναφορές προόδου με ποσοστά, προσπάθειες και οπτικές μπάρες.
- Αναλυτικές αναφορές με χρόνο ενασχόλησης, ολοκληρωμένες ενότητες, πρόσφατες προσπάθειες, συχνότερα λάθη και σημεία δυσκολίας.
- Adaptive learning με προτάσεις επανάληψης, συνέχισης ή προχωρημένης πρόκλησης ανάλογα με την επίδοση.

## Δομή περιεχομένου

Το εκπαιδευτικό υλικό έχει χωριστεί σε καθαρότερα αρχεία:

```text
content.py                 # public aggregator για server imports
curriculum/modules.py      # ενότητες, θεωρία, projects, δραστηριότητες
curriculum/questions.py    # τράπεζα ερωτήσεων quiz
```

## Εγχειρίδιο

Το εγχειρίδιο ανάλυσης και σχεδιασμού βρίσκεται στο:

```text
docs/manual.tex
docs/manual.pdf
```

Παραγωγή PDF με LaTeX:

```powershell
xelatex -output-directory docs docs/manual.tex
```

Υπάρχει επίσης βοηθητικό σενάριο για το βίντεο παρουσίασης:

```text
docs/presentation_script.md
```
