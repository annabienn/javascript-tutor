# Σενάριο βίντεο παρουσίασης

## 1. Εισαγωγή

Παρουσιάζουμε την εφαρμογή **JavaScript Tutor**, ένα εκπαιδευτικό λογισμικό για το Θέμα 2, με στόχο την εκμάθηση JavaScript από αρχάριους χρήστες.

Η εφαρμογή έχει υλοποιηθεί με Python, HTML, CSS, JavaScript και SQLite. Δεν βασίζεται σε έτοιμη πλατφόρμα ή no-code εργαλείο.

## 2. Εκκίνηση εφαρμογής

Τρέχουμε:

```powershell
python server.py
```

Μετά ανοίγουμε:

```text
http://127.0.0.1:8000
```

## 3. Αρχική σελίδα, εγγραφή και είσοδος

Δείχνουμε ότι η εφαρμογή ξεκινά από ξεχωριστή αρχική σελίδα σύνδεσης/εγγραφής. Το preview μαθημάτων υπάρχει ως αναδιπλούμενο μενού και δεν περιλαμβάνει quiz ή δραστηριότητες.

Κάνουμε εγγραφή ή είσοδο με email και κωδικό. Εξηγούμε ότι ο χρήστης αποθηκεύεται στη βάση και ότι η πρόοδος συνδέεται με τον λογαριασμό του.

## 4. Εκπαιδευτικές ενότητες

Παρουσιάζουμε ότι η εφαρμογή έχει εννέα ενότητες:

- Εισαγωγή στη JavaScript
- Μεταβλητές, τύποι και συνθήκες
- Συναρτήσεις και οργάνωση κώδικα
- Πίνακες, loops και επανάληψη
- Objects, JSON και μοντελοποίηση δεδομένων
- DOM, events και ασύγχρονη λογική
- Debugging και τελικό mini project
- Αποθήκευση στον browser και κατάσταση χρήστη
- Ολοκληρωμένη εφαρμογή, έλεγχος ποιότητας και τεκμηρίωση

Για κάθε ενότητα δείχνουμε ότι υπάρχουν μαθησιακός στόχος, οδηγός μελέτης, αναλυτική θεωρία, παραδείγματα κώδικα, δραστηριότητες, mini project και προσαρμοσμένη πρόταση.

## 5. Quiz αυτοαξιολόγησης

Απαντάμε σε ένα quiz ενότητας και πατάμε **Υποβολή απαντήσεων**.

Δείχνουμε σκορ, ποσοστό επιτυχίας, επεξήγηση για κάθε απάντηση και πρόταση επόμενης ενέργειας. Αναφέρουμε ότι κάθε ενότητα έχει οκτώ ερωτήσεις.

## 6. Επαναληπτικό quiz

Πατάμε **Επαναληπτικό Quiz**.

Εξηγούμε ότι αυτό συνδυάζει ερωτήσεις από όλες τις ενότητες. Στη συγκεκριμένη έκδοση περιλαμβάνει 45 ερωτήσεις.

## 7. Αναφορές προόδου

Δείχνουμε τις αναφορές:

- μέση επίδοση,
- αριθμό προσπαθειών,
- δραστηριότητες,
- ολοκληρωμένες ενότητες,
- χρόνο ενασχόλησης,
- επίδοση ανά ενότητα,
- πρόσφατες προσπάθειες,
- συχνότερα λάθη,
- σημεία δυσκολίας.

Εξηγούμε ότι τα δεδομένα προέρχονται από τη SQLite βάση και ανανεώνονται μετά από quiz ή δραστηριότητες.

## 8. Προσαρμοσμένη μάθηση

Δείχνουμε ότι η εφαρμογή αλλάζει πρόταση ανάλογα με την επίδοση:

- κάτω από 60% προτείνει επανάληψη,
- 60% έως 84% προτείνει συνέχιση,
- 85% και πάνω προτείνει πρόκληση.

## 9. Αποσύνδεση

Πατάμε **Αποσύνδεση** και δείχνουμε ότι ο χρήστης επιστρέφει στην αρχική σελίδα σύνδεσης/εγγραφής.

## 10. Κλείσιμο

Κλείνουμε αναφέροντας ότι η εφαρμογή καλύπτει την εκφώνηση με προγραμματιστική υλοποίηση, επαρκές περιεχόμενο, quiz, δραστηριότητες, αποθήκευση προόδου, αναφορές, adaptive learning και τεκμηρίωση σε LaTeX/PDF.
