# Σενάριο βίντεο παρουσίασης

## 1. Εισαγωγή

Παρουσιάζουμε την εφαρμογή **JavaScript Tutor**, ένα εκπαιδευτικό λογισμικό για το Θέμα 2, με στόχο την εισαγωγή αρχάριων χρηστών στη JavaScript.

Η εφαρμογή έχει υλοποιηθεί με Python, HTML, CSS, JavaScript και SQLite. Δεν βασίζεται σε έτοιμη πλατφόρμα ή no-code εργαλείο.

## 2. Εκκίνηση εφαρμογής

Δείχνουμε την εκτέλεση:

```powershell
python server.py
```

Μετά ανοίγουμε:

```text
http://127.0.0.1:8000
```

## 3. Δημιουργία μαθητή

Στο πάνω μέρος της εφαρμογής γράφουμε ένα όνομα μαθητή και προαιρετικά email, και πατάμε **Έναρξη**.

Εξηγούμε ότι εκεί δημιουργείται εγγραφή στη βάση δεδομένων και πλέον η πρόοδος του χρήστη αποθηκεύεται. Αν δοθεί το ίδιο email σε επόμενη είσοδο, η εφαρμογή επιστρέφει στο ίδιο προφίλ.

Δείχνουμε ότι η διεπαφή χωρίζεται σε αριστερό τμήμα προόδου και κύριο τμήμα μάθησης, ώστε ο χρήστης να βλέπει ταυτόχρονα περιεχόμενο και στατιστικά.

## 4. Εκπαιδευτικές ενότητες

Παρουσιάζουμε ότι η εφαρμογή έχει επτά ενότητες:

- Εισαγωγή στη JavaScript
- Μεταβλητές, τύποι και συνθήκες
- Συναρτήσεις και οργάνωση κώδικα
- Πίνακες, loops και επανάληψη
- Objects, JSON και μοντελοποίηση δεδομένων
- DOM, events και ασύγχρονη λογική
- Debugging και τελικό mini project

Για κάθε ενότητα δείχνουμε ότι υπάρχει:

- μαθησιακός στόχος,
- μαθησιακά αποτελέσματα,
- όροι-κλειδιά,
- θεωρία,
- παράδειγμα κώδικα,
- πρακτικές εργασίες,
- mini project,
- δραστηριότητες εξάσκησης με πεδίο απάντησης, άμεσο έλεγχο και ενδεικτική λύση,
- προσαρμοσμένη πρόταση.

## 5. Quiz αυτοαξιολόγησης

Απαντάμε σε ένα quiz ενότητας και πατάμε **Υποβολή απαντήσεων**.

Δείχνουμε:

- σκορ,
- ποσοστό επιτυχίας,
- επεξήγηση για κάθε απάντηση,
- πρόταση επόμενης ενέργειας.

Αναφέρουμε ότι κάθε ενότητα έχει έξι ερωτήσεις, ώστε η αυτοαξιολόγηση να μην είναι αποσπασματική.

Δείχνουμε επίσης μία δραστηριότητα εξάσκησης: γράφουμε απάντηση, πατάμε **Έλεγχος** και εμφανίζεται αν η απάντηση περιέχει τα βασικά σημεία.

## 6. Επαναληπτικό quiz

Πατάμε **Επαναληπτικό Quiz**.

Εξηγούμε ότι αυτό συνδυάζει ερωτήσεις από διαφορετικές ενότητες, ώστε να ελέγχεται συνολικά η κατανόηση και όχι μόνο μία θεματική περιοχή.

Στη συγκεκριμένη έκδοση το επαναληπτικό quiz περιλαμβάνει δεκατέσσερις ερωτήσεις.

## 7. Αναφορές προόδου

Στο αριστερό μέρος δείχνουμε:

- μέση επίδοση,
- αριθμό προσπαθειών,
- αριθμό δραστηριοτήτων,
- ολοκληρωμένες ενότητες,
- χρόνο ενασχόλησης,
- επίδοση ανά ενότητα,
- μαθησιακή διαδρομή.
- πρόσφατες προσπάθειες,
- συχνότερα λάθη,
- σημεία δυσκολίας.

Εξηγούμε ότι τα δεδομένα προέρχονται από τη SQLite βάση και ανανεώνονται μετά από κάθε προσπάθεια.

Αναφέρουμε ότι στη βάση αποθηκεύονται και οι προσπάθειες δραστηριοτήτων, όχι μόνο τα quiz.

## 8. Adaptive learning

Δείχνουμε τις διαφορετικές περιπτώσεις:

- χαμηλή επίδοση: πρόταση επανάληψης,
- μέτρια επίδοση: πρόταση συνέχισης,
- υψηλή επίδοση: προχωρημένη πρόκληση.

Τονίζουμε ότι η εφαρμογή δεν εμφανίζει απλώς στατικά μαθήματα, αλλά αλλάζει την πρόταση μάθησης ανάλογα με την απόδοση.

## 9. Βάση δεδομένων

Αναφέρουμε τους βασικούς πίνακες:

- users,
- module_progress,
- quiz_attempts,
- quiz_answers.
- activity_attempts.

Εξηγούμε ότι αποθηκεύονται απαντήσεις, σκορ, προσπάθειες, δραστηριότητες, επισκέψεις και χρόνος ενασχόλησης.

## 10. Κλείσιμο

Κλείνουμε λέγοντας ότι η εφαρμογή καλύπτει τις απαιτήσεις του Θέματος 2:

- προγραμματιστική υλοποίηση,
- τρεις ενότητες,
- quiz ανά ενότητα,
- επαναληπτική αξιολόγηση,
- βάση δεδομένων,
- αναφορές προόδου,
- προσαρμοσμένη μάθηση,
- εγχειρίδιο σε LaTeX.
