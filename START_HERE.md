# Οδηγίες για εκτέλεση

1. Άνοιξε φάκελο project.
2. Τρέξε:

```powershell
python server.py
```

3. Άνοιξε στον browser:

```text
http://127.0.0.1:8000
```

Σε Windows μπορείς επίσης να κάνεις διπλό κλικ στο `run_windows.bat`.

Δεν χρειάζεται εγκατάσταση βιβλιοθηκών. Η εφαρμογή χρησιμοποιεί μόνο Python standard library και SQLite.
